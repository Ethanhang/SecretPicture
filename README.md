# SecretPicture
基于LSB算法实现了对水印图片，文字等信息的隐藏和提取的DEMO，原理是通过读取和替换原始图像像素中RGB通道值的最低有效位(LSB)，实现了将水印图片和文字二进制信息嵌入到原图中的功能，功能包括嵌入水印，提取水印；原始图片支持常用的图片格式,包括.png，.jpg，.bmp等。
